#try-workflow

Gulpfiles for CoffeeScript, Browserify, Watchify, Stylus, Pug (Jade)
___

###Install

`npm install`

###Run development mode
Starts gulp-server-livereload and watches `.coffee` (through Watchify), `.styl`, `.pug`

`npm start` or `gulp`

###Build only

`gulp build`
